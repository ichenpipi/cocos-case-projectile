# Projectile

一个使用 Cocos Creator 开发的炮弹抛射示例项目。

项目核心主题为抛射体运动（Projectile Motion）在游戏开发中的实践。

开源仓库：[https://gitee.com/ifaswind/cocos-case-projectile](https://gitee.com/ifaswind/cocos-case-projectile)
